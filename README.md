#  AI-Based Rock Paper Scissors
Created an AI-based Rock Paper Scissors game that adapts to player behavior using ML.
An **Intelligent Rock–Paper–Scissors game** that doesn’t play randomly.  
This AI **learns from the player’s behavior**, detects patterns, and adapts its strategy using **Machine Learning concepts** like frequency analysis and Markov Chains.

> 🧠 After a few rounds, the AI starts predicting your next move!

## 🎯 Project Objective

Traditional Rock–Paper–Scissors games rely on randomness.  
This project introduces **adaptive intelligence**, where the system:

- Observes player choices
- Learns decision patterns
- Predicts future moves
- Counters strategically

The goal is to demonstrate **practical ML concepts in a fun, interactive way**, using only frontend technologies.

## 🧩 How the AI Works

The intelligence is **rule-based ML logic**, inspired by real machine learning strategies:

- **Pattern Analysis**  
  Tracks sequences in recent player moves

- **Frequency Learning**  
  Calculates probability distribution of player choices

- **Markov Chain Logic**  
  Predicts the next move based on transition patterns

- **Adaptive Strategy**  
  Counters predicted moves with ~70% confidence

- **Anti-Exploitation**  
  Adds ~30% randomness so the AI doesn’t become predictable

- **Training Phase**  
  Requires **5+ games** before predictions activate

> This balance ensures the AI feels smart but still fair.
